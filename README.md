# DOOM風FPSゲーム

## 実行環境の必要条件
* Python >= 3.10
* pygame >= 2.5
* minipy3dr >= 0.5.0 (`pip install minipy3dr`)

## ゲームの概要
* MiniPy3DR(pygame製の3Dレンダラ)のFPSツールキット `minipy3dr.fps` を使ったDOOM風の3D FPSゲームで、マップ探索やアイテム収集を行いながら進める
* 矢印キーでキャラクターを操作し、マップ内のギミックを解き明かす

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画、FPSの基本システムの構築

### 分担追加機能
* アイテム機能（担当：吉荒倖汰）：各種アイテムの配置と以下の特殊ギミックの実装
  * 全アイテムの回転アニメーションと、取得時のパーティクル・フラッシュ演出
  * ゴールド(G)およびレアアイテム(R)の配置とスコア加算処理
  * 鍵(K)を取得後、扉(D)の付近でEキーを押して開く進行ギミック
  * 赤色アイテム(U)による30秒間のスコア2倍・弾薬自動回復のパワーアップ機能

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
